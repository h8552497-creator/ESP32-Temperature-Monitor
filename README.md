# TM2841 - ESP32 Temperature Monitor

A simple and beginner-friendly temperature monitoring project built using ESP32, DHT11, and an OLED display.

## Features

* Real-time temperature monitoring
* OLED display output
* Easy hardware setup
* Beginner-friendly code
* Low-cost project

## Components

* ESP32 DevKit V1
* DHT11 Sensor
* SSD1306 OLED Display
* Jumper Wires

## Applications

* Home temperature monitoring
* Educational projects
* IoT learning
* Sensor interfacing practice
* Environmental monitoring
